Key Features

Multiple AI Providers

  • • Google Vertex AI (Gemini 1.5 Pro)
  • • OpenAI (GPT-4)
  • • Azure OpenAI (GPT-4)
  • • Ollama (Local deployment)
  • • OpenAI-Compatible APIs

Fast Data Sync

Parallel data fetching for quick synchronization with your Okta tenant. Local SQLite storage ensures rapid query responses.

Data Transparency

Full visibility into stored data including Users, Groups, Applications, Factors, and Policies. Inspect data using SQLite Browser.

Security First

Local Data Storage

Your Okta data never leaves your machine. All data is stored locally in a SQLite database.

API Control

You control the Okta API token, its permissions, and network restrictions. Full control over access.

Data Privacy

  • ✓ Only queries sent to LLMs
  • ✗ No user data sent
  • ✗ No organizational data sent
  • ✗ No synced database contents sent
